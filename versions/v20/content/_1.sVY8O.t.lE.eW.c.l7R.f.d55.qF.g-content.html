Diagram Figure R.10 - Associating EntityRoles with an Entity
An EntityRole is an abstract class that extends the Role class to define the concept of various types of roles for Entities. Following Figure shows the relationships between an Entity, an EntitySpecification, a RoleSpecification, and an EntityRole.
RoleSpecifications can be used to assign different types of Roles to an Entity. This is done using the SpecifiesRoles association.
