Class SecVulnerabilityMetricValueDefn

A standard value for a defined scoring metric (SecVulnerabilityScoringMetricDefn). Each metric has multiple instances which represent multiple values.

Notes: For example, Base Metric Access Vector has three pre-defined values - L/Local, A/Adjacent Network, and N/Network.

String  description

A narrative that explains the purpose of the of the SecVulnerabilityMetricValueDefn.

Notes: As an example, L (local) is defined by NIST per the CCSS Report 7502 as "A vulnerability actively exploitable with only local access requires the exploiter to have either physical access to the vulnerable system or a local (shell) account".
