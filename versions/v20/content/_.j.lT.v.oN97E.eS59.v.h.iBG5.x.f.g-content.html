Common Domain::Party Privacy ABE::Party Privacy Profile Type ABE
Class PartyPrivacyProfileType

A PartyPrivacyProfileType is a type of PartyProfileType.
A PartyPrivacyProfileType specifies for a Party playing a type of PartyRole (PartyRoleSpecification) such as Customer or User and possibly restricted to a context such as a set of ProductOfferings, all the available options that the Party may choose for each information concerned by the Privacy.
A PartyPrivacyProfileType may specify Privacy for several information using PartyPrivacyProfileTypeCharacteristic.
