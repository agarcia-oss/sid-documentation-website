Activities have statuses. One possible set of statuses is shown in figure 6, based on those used in BPML [BPML]
The arrows show the allowable state transitions. Note that the Activating, Aborting and Completing statuses will be short lived and in effect are notifications of a transition, which will be useful in implementing an event model.
