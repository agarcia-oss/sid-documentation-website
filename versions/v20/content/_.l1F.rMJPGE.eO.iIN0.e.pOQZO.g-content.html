Diagram Figure Me.10 - Metric Measure Definition Groups
The following Figure shows how MetricDefMeasures and MetricMeasures may be grouped when needed.

A MetricDefMeasureGroup is a set of MetricDefMeasures, usually reported in the same message by the equipment or the EMS.
A MetricMeasureGroup is a set of MetricMeasures, usually reported in the same message by the equipment or the EMS.

For example, a server will typically report CPU related metrics in a single message. These metrics may include: Maximum CPU usage, Average CPU usage, System CPU usage and User CPU usage.
Mobile RAN may group related metrics by functional aspects in a technology context such as: LTE Intra-frequency handovers, UMTS Radio Access or NodeB Utilization.
Standard protocols like MTOSI or 3GPP will pack the Metric Group in a single message either the same line in a CSV file or within XML tags when the data format is XML.
