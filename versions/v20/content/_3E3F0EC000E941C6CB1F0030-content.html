Product Domain::Product Usage ABE::Product Usage Spec ABE
Class CompositeProductUsageSpec

A detailed description of the usage of a Product, specified by CompositeProductSpecification. It is formed by aggregating other ProductUsageSpec(s), which may be either Composite or Atomic ProductUsageSpec(s) or Atomic ProductUsageSpec(s).
