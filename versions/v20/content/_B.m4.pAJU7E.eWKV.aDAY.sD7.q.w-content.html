Diagram Figure R.12 - ManagementInfo Class Hierarchy
ManagementInfo is an abstract class that is used to represent entities that represent management information obtained in a managed environment. Specifically, in the process of managing an entity, information of various forms is created. This class is the superclass for defining different types of management information. Its main subclasses are shown in the Figure below.
The design of how management information is obtained uses this class to gather specific types of information, and the ManagementMethodEntity class to obtain the desired data using an appropriate method. This combination of class hierarchies enables different management methods (e.g., CLI vs. SNMP) to be used to gather different types of management data, and is unique to the SID information model.
