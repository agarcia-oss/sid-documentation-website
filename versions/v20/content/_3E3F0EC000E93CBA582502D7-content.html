Common Domain::Location ABE::Local Place ABE
Class GridLocalLocation

A LocalLocation that is defined using Grid coordinates that relate back to a LocalGridCoordinateSystem
This would be used when locations are not defined in inches or millimeters but rather in terms of things like floor tile positions.
The grid units are defined in the LocalCoordinateSystem.
