Common Domain::Test ABE
Enumeration TestMode

An indication of whether the test is "PROACTIVE" and running at the time and entity is created and activated, or whether the test is "ONDEMAND"

Enumeration Literals
 ONDEMAND

ONDEMAND implies the test was started explicitly via an operation
 PROACTIVE

PROACTIVE implies the test was start implicitly when an entity was created and actived, vs. explicitly via an operation
