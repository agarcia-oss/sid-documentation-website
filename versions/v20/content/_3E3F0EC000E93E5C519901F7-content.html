Class ResourceFacingServiceSpecAtomic

This is an abstract base class for defining ResourceFacingServiceSpecifications that do not have any subordinate ResourceFacingServiceSpecifications. In other words, a ResourceFacingServiceSpecAtomic is a stand-alone ResourceFacingServiceSpecification, and doesn't require any supporting ResourceFacingServiceSpecifications to define the shared characteristics (i.e., attributes, relationships, and constraints) of any ResourceFacingServices that it serves as a template for.
