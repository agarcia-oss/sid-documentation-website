Class UrbanPropertySubAddress

An UrbanPropertySubAddress is used for addressing within a property in an urban area (country properties are often defined differently).. It may refer to a building, a building cluster, or a floor of a multistory building.This is an example of a concrete Geographic Location entity.

Attributes
String  buildingName

A well known name for a building.

Notes: e.g. Telstra house, Parliament house
String  levelNr

A number used where a level type may be repeated e.g. BASEMENT 1, BASEMENT 2.

Notes: some level Types do not require a level Nr e.g. BASEMENT, LOWER GROUND, MEZZANINE, UPPER GROUND
String  levelType

The level types within a building.

Notes: e.g. BASEMENT, FLOOR, LOWER GROUND, LEVEL, MEZZANINE, UPPER GROUND
String  privateStreetName

Private streets internal to a property (e.g. a university) may have internal names that are not recorded by the land title office.
String  subUnitNr

The discriminator used for the subunit.

Notes: often just a sumple number e.g. FLAT 5, may also be a range
String  subUnitType

The type of subunit.

Notes: e.g.BERTH, FLAT, PIER, SUITE, SHOP, TOWER, UNIT, WHARF
