Class CommonPlatformEnumeration

Attributes
Hardware  _hardware

String  cpeName

A CPE Name is represented by a Well Formed Name (WFN) that is bound to a URI, which provides backward compatability with CPE 2.2, and a formatted string. For further details on CPE refer to NIST IRs 7695-7698.

Notes: "For example, the following represents a CPE WFN definition of Microsoft Internet Explorer 8.0.6001 Beta:
wfn:[part=""a"",vendor=""microsoft"",product=""internet_explorer"", version=""8\.0\.6001"",update=""beta""]. An example of the formatted string binding is: cpe:2.3:a:microsoft:internet_explorer:8.0.6001:beta:*:*:*:*:*:* and URI binding is: cpe:/a:microsoft:internet_explorer:8.0.6001:beta."
