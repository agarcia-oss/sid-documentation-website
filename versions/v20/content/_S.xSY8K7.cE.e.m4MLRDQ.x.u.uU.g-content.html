Diagram Figure [PrD-02] Product ABEs Level 1 & 2
The diagram below lists the Product Domain's ABEs Level 1 & 2.

[] Network Product ABE
• The Network Product ABE contains the NetworkProduct description and relationships.
[] Product Price ABE
• The Product Price ABE describes all the prices applied to Products.
[] Usage Volume Product ABE
• The Usage Volume Product ABE contains the UsageVolumeProduct and all the entities specific to this type of product.
[] Loyalty Program ABE
• The Loyalty Program ABE contains all entities needed to instantiate a loyalty program for a specific customer such as LoyaltyProgram and LoyaltyAccount.
[] Loyalty Program Specification ABE
• The Loyalty Program Specification ABE contains all entities specifying a loyalty program such as LoyaltyProgramProdSpec and LoyaltyRule.
[] Network Product Spec ABE
• The Network Product Spec ABE contains the NetworkProductSpec description and relationships.
[] Usage Volume Product Spec ABE
• The Usage Volume Product Spec ABE contains the UsageVolumeProductSpec and all the entities specific to this type of product.
[] Product Action ABE
• The Product Action ABE contains Business Entities describing possible actions for ProductSpecifications, ProductOfferings and ProductConfigSpecs.
[] Product Usage Spec ABE
• The Product Usage Spec ABE specifies types of usage of ProductSpecifications.
[] Product Catalog ABE
• Product Catalog ABE is a collection of entities that represent the specification of ProductCatalogs and catalogs described by them that contain ProductOfferings and their associated ProductOfferingPrices.
[] Product Offering Price ABE
• The Product Offering Price ABE describes all the prices related to ProductOfferings.
[] Product Offering Price Rule ABE
• The Product Offering Price Rule ABE describes all rules related to prices concerning ProductOfferings.
[] Pricing Logic Algorithm ABE
• The Pricing Logic Algorithm ABE contains entities representing an instantiation of an interface specification to external rating function (without a modeled bahavior in SID).
[] Product Placement ABE
• The Product Placement ABE gives a description of how ProductOfferings are marketed, such as via direct mail. A ProductPlacement contains qualitative statements, or assessments that describe such things as market penetration, product strategy, and so forth.
[] Product Promotion ABE
• The Product Promotion ABE describes the publication of a ProductOffering to encourage Parties to buy, rent, or in some other way procure the offering.
