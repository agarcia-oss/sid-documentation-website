Diagram Figure PF.04 - Performance Specification Interval
One entity not included in the previous Figure, PerformanceSpecificationInterval, specifies the frequency at which Performance is calculated is shown in following Figure. The figure includes the PerformanceSpecIntervalConversion entity which specifies how one conversion interval, such as hourly interval, is calculated from a five-minute interval. Also shown in the figure is an association that enables customer-specific PerformanceSpecification to be created. These and the other entities in the figure depict the complete definition of a PerformanceSpecification.
The PefromanceIntervalSpecification and its related PerformanceSpecIntervalConversion entity also provide for an extensible model by allowing any type of interval to be defined in addition to those that currently exist.
