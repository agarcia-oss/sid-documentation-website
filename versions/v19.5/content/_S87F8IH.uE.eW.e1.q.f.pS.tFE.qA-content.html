Class Currency

Represent the currency in reality. Currency uses 1 or more CurrencyUnits. For example, Currency (code=CNY, symbol= ï¿¥) use three CurrencyUnits (Yuan, Jiao, Fen). Currency is associated to CurrencyExchangeRate with CurrencyUsedAsSource and CurrencyUsedAsDestination. Currency can be used in different Countrys.

Attributes
String  code

Represents the code of the currency in the usage for convenience. Generally it is made up of three capital English letters, the first two of which represent the country and the last of which represents the name of the currency. For example, USD represents United State of America Dollar. Usually ISO 4217 is used.
