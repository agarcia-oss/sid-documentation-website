Service Domain::TIP Service Management ABE
Class ServiceDefinition

Represents a type of Service Specification introduced for the purposes of Service Fulfillment. It defines ALL the ServiceSpecCharacteristics (SSCs) that must be used to create corresponding Service instances. This includes the SSCs which are:
globally set (the corresponding values are defined only in ServiceTemplates and are sometimes designated as “invariant”) and
set per instance (since the ServiceDefinition does not contain any value for such SSCs, the corresponding values can be defined only over the Activation Interface and are sometimes designated as “variant”)
