Diagram Figure Cal.05 - Calendar Detail
The calendar entity has defaults to be applied when a user receives and views CalendarEntries. For example, it has a time zone reference that allows it to convert entries from different time zones to a standard time zone.
It also has a calendrical System reference so that it knows the format to present the calendar in. For example, the Gregorian calendar is used in “western” countries. Other calendrical systems in use today [calendrical systems] include the Islamic, Persian (Jalali), Hebrew and Chinese calendrical systems.
