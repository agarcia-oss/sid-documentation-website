Diagram Figure L.10b - Geographic Location Symbology
To display features on a map we need to define the display format. The following Figure shows how each type of Geographic Location can be assigned a symbology definition.
There are two options for naming of GeographicLocations. If a GeographicLocation type only has a single name, then the name should be included as a simple attribute (e.g. Country). If the GeographicLocation type is likely to have names that change over time, or has alternative names (e.g. native names), then the GeographicLocartionName entity should be used (e.g. Administrative Area).
