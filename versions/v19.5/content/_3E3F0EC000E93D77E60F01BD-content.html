Common Domain::Policy ABE
Policy Structure ABE

The Policy ABE contains entities that are used to represent the various components of a Policy. These components center around the concept of a PolicyRule, which is a triplet defined as:

{event, condition, action}

This has the semantics of: "ON receipt of an event, IF the condition clause is TRUE, THEN execute a set of actions. A novel feature of this ABE is that the same triplet can be used to form condition and action clauses. This ABE also contains the ability to group and nest PolicyRules and their components.
