Diagram Figure Ppr.01 - Reminder about RootEntityType and RootEntity
The Party Privacy Management process manages privacy information and agreement with party.
Therefore, it needs to specify the information concerned by Privacy rules and the Privacy rules themselves.

The following Figure depicts how all information of the SID model might be addressed.
Each SID Entity corresponds to an instance of RootEntityType and each attribute of an Entity corresponds to the use (RootEntityTypeCharUse) of a CharacteristicSpecification.
Each possible value of a CharacteristicSpecification (CharacteristicSpecValue) might be used or not in the context of a RootEntityType (RootEntityTypeCharValueUse).
Each instance of a SID Entity corresponds to an instance of RootEntity and the value of its attributes are carried by an instance of CharacteristicValue.
