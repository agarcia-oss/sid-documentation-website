Class SecVulnerabilityMetricDefnMbr

A type of SecVulnerabilityScoringAlgorithmMember that indicates the score of the associated SecurityVulnerabilityMetric should be used at this point in the algorithm.

Notes: For example, the score for the Access Vector should be used at this point in the algorithm Exploitability score = 20 * Access Vector score * Authentification score * Access Complexity score.
