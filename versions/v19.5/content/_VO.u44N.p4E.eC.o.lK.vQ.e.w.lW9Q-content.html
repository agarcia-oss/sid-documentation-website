Enterprise Domain::Workforce ABE::Workforce Resource ABE
Class WorkforceResourcePool

Represents a resource pool as a set of resources (both physical and logical) available for assignment to work tasks. For example, the Dallas pool for fiber rollout for a weekday contains X number of trucks, Y number of test kits, etc.
