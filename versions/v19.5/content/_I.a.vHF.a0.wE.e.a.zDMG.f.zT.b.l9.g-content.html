Attachment represents a group of classes that support an attachment like image, video and document for any RootEntity or BusinessInteraction. The following Figure shows the UML diagram of the objects in this fragment.
[] Attachment
This is the class to represent an attachment like document, image or video to any RootEntity or BusinessInteraction.
This class is concrete and it can be used without a sub-class. However, one can extend this class to add more specific behavior of an entity attachment. The EntitySpecificationDocumentation and EntitySpecificationImage are examples of such extension.
[] EntitySpecificationDocumentation
This is the extended class representing a documentation attachment for an entity specification.
[] EntitySpecificationImage
This is the extended class representing an image attachment for an entity specification.
