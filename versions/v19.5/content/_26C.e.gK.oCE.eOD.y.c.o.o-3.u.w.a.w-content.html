Diagram Figure CP.02a - Configuration Specification Relationship
The following Figure shows additional details about the relationships that can be defined among ConfigurationSpecifications.
A type of ConfgSpecRelationship that represents a group of ConfigurationSpecifications that are used to configure a ConfigurationSpecification that is made up of other ConfigurationSpecfications. For example, the ConfigurationSpecification for a connection may be composed of the specification for the connection as well as the specification for connection termination points that are assigned to the connection.
The configSequence attribute specifies the order in which a related ConfigurationSpecification that is part of a group (composite) is to be configured. The configAction attribute represents a prescribed action that is taken for a ConfigurationSpecification that is part of a group (composite). For example, "assign" a connection termination point to a connection. The configQuantity attribute is a number that represents how many of a related ConfigurationSpecification in a group (composite) are to be configured.
