Diagram Figure PR.26 - Types of Cards
The Figure below illustrates the superclasses for defining different types of Cards.
MemoryCards are dedicated to providing additional memory for use by other components of a Resource. NetworkCards are dedicated to providing networking functions, such as routing and forwarding. SystemCards are dedicated to providing system functions (e.g., controller cards). Finally, the UnknownCard object is used as a generic placeholder to represent Cards that are known to exist but are not yet identifiable via Software means.
