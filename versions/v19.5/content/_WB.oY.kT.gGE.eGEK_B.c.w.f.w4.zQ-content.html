Class CapacityDemand

The amount of capacity that is required, planned to be consumed or has been consumed.

Attributes
AppliedCapacityDemand  _appliedCapacityDemand
ApplicableTimePeriod  _capacitySchedule
«baseType» Quantity  capacityDemandAmount

A value and units that define the CapacityDemand, such as 10000 ea, 10B Mb.

Instance values are mutually exclusive with From and To capacityDemandAmounts and range interval.
«baseType» Quantity  capacityDemandAmountFrom

The low range value that a CapacityDemand can take on.
«baseType» Quantity  capacityDemandAmountTo

The upper range value that a CapacityDemand can take on.
String  capacityDemandType

An indicator that specifies whether the capacity demand is planned, actual or required by.
String  priority

The relative importance of the CapacityDemand.
String  rangeInterval

An indicator that specifies the inclusion or exclusion of the capacityDemandAmountFrom and capacityDemandAmountTo attributes.

Possible values are "open", "closed", "closedBottom" and "closedTop".
