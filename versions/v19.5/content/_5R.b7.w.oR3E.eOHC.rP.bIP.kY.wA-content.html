In some cases the determination of which ConfigurationSpecification should be used is dependent upon a number of variables. The following Figure shows the use of characteristics to represent these variables.
The ConfigurationSpecificationFurtherDefinedBy relationship indicates constraints used to determine char value that is dependent on other char values.
The ConfigurationSpecDeterminedBy relationship added so constraints can be used to "select" a ConfigurationSpec.
