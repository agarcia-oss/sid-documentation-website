Class ServiceStateInfo

ServiceStateInfo is an abstract base class for representing different types of management information that describes the state of a particular Service. This can be a CustomerFacingService or a ResourceFacingService.
Subclasses of this class include classes to represent Faults, Outages, Alarms, the usage of this Resource, and its current operational state.
