Enterprise Domain::Workforce ABE::Workforce Schedule ABE
Class WorkforceResourceAssignment

Represents an assignment of a particular resource for particular work in a specific timeframe. For example, assignment of a fiber test kit (or, additional example, a role of a test kit) to a fiber rollout work for a date.
