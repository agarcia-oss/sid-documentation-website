Service Domain::Service Performance ABE::Service Level Spec ABE
Class ServiceLevelSpecApplicability

The time of day or days during which a ServiceLevelSpecification, ServiceLevelObjective, or ServiceLevelSpecConsequence is relent or not.

Attributes
String  unapplicableApplicableCode

Indicator that specifies whether an instance represents applicability or un-applicability.
«baseType» TimePeriod  unappAppDuration

A period of time representing applicability or un-applicability, for example from 8AM to 12PM.

For an instance either unappAppDuration or unappAppDays must have a value.
String  unappAppDays

A day or days representing applicability or un-applicability, for example 2,3 represents Monday and Tuesday.

For an instance either unappAppDuration or unappAppDays must have a value.
