Service Package ABE

The Service Package ABE defines the concept of bundling a set of different CustomerFacingServiceSpecs to meet the functionality specified by one or more ProductSpecifications. This enables the shared characteristics of these CustomerFacingServices to be specified, so that multiple Products can be built from their associated ProductSpecification.
