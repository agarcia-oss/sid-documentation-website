PhysicalResource ABE

The Resource ABE contains entities that are used to represent the various aspects of a Resource.

PhysicalResource entities are sets of entities that represent the physical aspects of a Resource. This is important to enable the SID to build a set of reusable managed entities that can be used to model the physical aspects of different types of Resources.
