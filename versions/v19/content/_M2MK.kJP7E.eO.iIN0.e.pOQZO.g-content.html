Figure Me.12 - Monitored Metric Object Criterion shows the different types of MonitoredMetricObjectCriterion.

A MonitoredMetricObjectCriterion defines the scope of monitored objects for which a MeasurementJob is executed.
The scope is either defined:
• by a filter on a Class (sub-class MonitoredMetricClassCriterion)
• or by a list of instances (sub-class MonitoredMetricInstanceCriterion).

A MonitoredMetricClassCriterion is a type of MonitoredMetricObjectCriterion.
It defines the scope of monitored objects for which a MeasurementJob is executed.by a filter on a class through the relationship with RootEntityType.

A MonitoredMetricInstanceCriterion is a type of MonitoredMetricObjectCriterion.
It defines the scope of monitored objects for which a MeasurementJob is executed by a list of instances through the relationship with one or several RootEntity instances.
