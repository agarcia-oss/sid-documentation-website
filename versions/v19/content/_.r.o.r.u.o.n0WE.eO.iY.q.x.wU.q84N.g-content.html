Class MonitoredMetricClassCriterion

A MonitoredMetricClassCriterion is a type of MonitoredMetricObjectCriterion.
It defines the scope of monitored objects for which a MeasurementJob is executed.by a filter on a class through the relationship with RootEntityType.
