Market_Sales Domain::Market Segment ABE::Market Statistic ABE
Class MarketStatistic

A categorization of performance measures by MarketSegment. Performance is measured for the ServiceProvider and a ServiceProvider's Competitors in the market place.

Attributes
String  ID

A unique identifier for the MarketStatistic.
String  description

A narrative that explains what the MarketStatistic is.
String  type

A division of MarketStatistic into groups because they exhibit common characteristics. For example, market penetration, churn, retention, market share, and so forth.
String  classification

A specification of what the MarketStatistic represents, for example a ProductProgram, ProductPromotion, ProductOffering, Branch, and so forth.
String  level

Describes the place the MarketStatistic holds in a hierarchy of MarketStatistics. For example, director, executive, external, and so forth.
«baseType» Quantity  amount

An number use to value the MarketStatistic, such as a percent. Mutually exclusive with the currencyAmount attribute.
«baseType» Money  currencyAmount

A current amount used to value the MarketStatistic. Mutually exclusive with the amount attribute.
«baseType» TimePeriod  validFor

The period during which the MarketStatistic is applicable.
