Class ProtocolTransferData

Defines the information required for a data transfer. It contains information needed for the data transfer, but it does not contain result information, i.e. the list of file produced is not part of this object nor of its descendants.

This class acts as a base class for all types of possible data transfers even if currently only File Transfer is supported.
