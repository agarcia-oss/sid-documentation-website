Class IPsecVPNService

A Internet Protocol Security (IPsec) VPN Service (IPsecVPNService) refers to the process of creating and managing VPN connections or services using an IPsec protocol suite. It is a secure means of creating VPN that adds IPsec bundled security features to VPN network packets. IPsec VPN is also known as VPN over IPsec.
