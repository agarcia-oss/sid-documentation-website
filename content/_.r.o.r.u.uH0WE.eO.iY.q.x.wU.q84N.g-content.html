Common Domain::Metric ABE::Metric Monitoring ABE
Class MonitoredMetricObjectCriterion

A MonitoredMetricObjectCriterion defines the scope of monitored objects for which a MeasurementJob is executed.
The scope is either defined:
•    by a filter on a Class (sub-class MonitoredMetricClassCriterion)
•    or by a list of instances (sub-class MonitoredMetricInstanceCriterion).
