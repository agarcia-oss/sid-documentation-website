Enterprise Domain::Enterprise Risk ABE::Enterprise Security ABE::Security Vulnerability ABE
Class SecurityVulnerabilityCategoryType

Identifies the type of SecurityVulnerability for the purposes of vulnerability scoring.



Notes: The three types defined include software flaws, security configuration issues, and software feature misuses.

Attributes
SecVulnerabilityScoringSystem  securityVulnerabilityScoringSpec
SecurityVulnerabilityCategoryAssignment  securityVulnerabilityCategory
String  name

Name assigned to the security vulnerability category type.

Notes: Software Flaw, Security Configuration Issue, and Software Feature Misuse.
String  description

Defines the intention behind the security vulnerability category type.

Notes: For example, a software flaw vulnerability (category type) is defined as an unintented error in the design or coding of software. A security configuration issue vulnerability (category type) is defined as a security configuration setting that negatively affects the security of the software. And a software feature misuse vulnerability (category type) is defined as a vulnerability exposed by a feature that provides an avenue to compromise the security of a system.
